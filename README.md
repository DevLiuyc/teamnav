# teamnav

# 一、介绍
本项目旨在为中小企业团队打造一款专属的团队网址导航系统，专注于提供便捷的网址导航服务以及强大的项目文档和看板管理功能，以显著提高团队工作效率。

# 二、功能说明
1. 网址导航功能，支持搜索
2. 网址导航管理端功能，导航分类和导航卡片的增删改查功能
3. 支持多用户登录、用户管理、权限管理

# 三、技术栈
整体功能比较简单，保持整体功能和部署的简单性，使用技术栈如下：

前端： Vue + BootStrap

后端： SpringBoot + SpringDataJpa + Thymeleaf + SpringSecurity

数据库：H2数据库
